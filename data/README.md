# Data

Folder for data storage (generated by Python scripts) and reference in analysis and visualization.

Codebook
| Variable Name | Description | Type | Values / Notes |
|---------------|-------------|------|----------------|
| game_pk | Unique identifier for each game | Categorical | 6 digit ID code |
| batter | Unique identifier for each batter | Categorical | 6 digit ID code |
| pitcher | Unique identifier for each pitcher | Categorical | 6 digit ID code |
| launch_speed | Speed of the ball off of the bat | Numeric | Measured in miles per hour | 
| launch_angle | Vertical angle of the ball off of the bat | Numeric | Measured in degrees; negative = downward, positive = upward |
| hit_distance | Distance or estimated distance ball travelled in air before hitting the ground | Numeric | Measured in feet; estimated for home runs that landed in the stands |
| hit_location_x | x coordinate | Numeric | location |
| hit_location_y | y coordinate | Numeric | location |
| pitch_speed | Speed of the pitch at release | Numeric | Measured in mph |
| release_spin_rate | Rotation of the ball at release | Numeric | Measured in rotations per minute (rpm) |
| xBA | Expected batting average | Numeric (proportion) | Range 0-1; Statcast metric that measures the probability that a ball with the given launch speed and angle will result in a hit |
| events | Result of the play | Categorical | single, double, triple, home_run, field_out, force_out, grounded_into_double_play |
| team | Batting team name | Categorical | 3-letter MLB team name abbreviations (e.g., BOS = Boston Red Sox, NYY = New York Yankees) |
| hit | Records if a hit occurs | Binary | 0-1; 0 = out, 1 = hit |
